# DevAPI Hub

DevAPI Hub is a full-stack API development and testing platform built with the MERN Stack and TypeScript. It allows developers to create, send, and manage HTTP requests through a clean and intuitive interface.

## Features

- User Authentication (JWT)
- Send GET, POST, PUT, PATCH and DELETE requests
- Add Headers and Query Parameters
- Bearer Token Authentication
- JSON Request Body Support
- Environment Variables
- Request History
- Save Request Collections
- Import & Export Collections
- Light and Dark Theme
- Responsive User Interface

## Tech Stack

### Frontend
- React
- TypeScript
- CSS
- Axios

### Backend
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Clone the repository

```bash
git clone https://github.com/ashwanijangra/Dev-API-Hub.git
```

### Install dependencies

#### Client

```bash
cd client
npm install
npm run dev
```

#### Server

```bash
cd server
npm install
npm run dev
```
